## 中文

### 关于

ZhangNote 是一款现代化的 AI 驱动 Markdown 编辑器，专为知识工作者、研究人员和笔记爱好者设计。

### 主要功能

- **Markdown 编辑** - 实时预览、语法高亮、代码块、LaTeX 数学公式
- **AI 智能** - 对话助手、内容润色、RAG 知识检索
- **可视化** - 知识图谱、思维导图生成
- **学习工具** - 试卷生成
- **MCP 支持** - 扩展 AI 工具能力

### About

ZhangNote is a modern AI-powered Markdown editor designed for knowledge workers, researchers, and note-taking enthusiasts.

### Key Features

- **Markdown Editing** - Live preview, syntax highlighting, code blocks, LaTeX math
- **AI Intelligence** - Chat assistant, content polish, RAG knowledge retrieval
- **Visualization** - Knowledge graph, mind map generation
- **Learning Tools** - Quiz generation
- **MCP Support** - Extended AI tool capabilities
